# TIMELAB Dashboard

Dashboard React/Vite conectado a Google Sheets mediante Google Apps Script.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicación en GitHub

Sube el contenido de esta carpeta a tu repositorio. Luego puedes desplegarlo en Vercel, Netlify o usar GitHub Pages con un workflow de build.
